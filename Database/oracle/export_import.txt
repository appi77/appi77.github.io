1. 클라이언트 쪽 풀백업 (Export)
​설치가 완료된 상태에서 "언제든 복원 가능한 상태"를 만들려면, DBeaver를 통해 본인 PC(클라이언트)에 덤프 파일을 생성해야 합니다.
​DBeaver 설정 및 실행:
​오라클 클라이언트 연결: DBeaver 상단 메뉴 Window -> Preferences -> Connections -> Drivers -> Oracle에서 Local Client 경로가 본인 PC의 오라클 bin 폴더로 지정되어 있어야 합니다. (안에 exp.exe가 있어야 함)
​백업 실행: - 대상 계정 우클릭 -> [Tools] -> [Dump Database]
​Export Mode: User (본인 계정 선택)
​Output: 본인 PC의 안전한 경로 지정 (예: C:\Backup\package_full.dmp)
​[Start] 클릭
​2. 복원 (Import)
​오브젝트를 삭제한 빈 상태에서 백업해둔 .dmp 파일을 다시 밀어넣는 과정입니다.
​복원 실행:
​대상 계정 우클릭 -> [Tools] -> [Restore Database]
​Input file: 이전에 백업한 package_full.dmp 선택
​Settings: From User와 To User를 본인 계정으로 동일하게 지정
​[Start] 클릭
